---

# 📖 Descripción

MG Dealer Management System (MG-DMS) es un proyecto de portfolio desarrollado con el objetivo de demostrar el trabajo de un Analista Funcional durante el ciclo completo de análisis, documentación y diseño de un sistema de gestión para concesionarias.

El proyecto está basado en procesos reales del negocio automotor, documentando desde el ingreso de un vehículo a la agencia hasta la entrega de la documentación al nuevo propietario.

Este repositorio no busca únicamente mostrar documentación, sino reflejar una metodología de trabajo profesional similar a la utilizada por consultoras y equipos de desarrollo de software.

---

# 🎯 Objetivos del Proyecto

- Comprender el funcionamiento integral de una concesionaria.
- Identificar oportunidades de mejora en los procesos actuales.
- Documentar el negocio utilizando buenas prácticas de Análisis Funcional.
- Diseñar un sistema de gestión orientado a optimizar la operación diaria.
- Mantener trazabilidad entre el negocio, los requerimientos y la futura solución.

---

# 🚘 Alcance del Proyecto

El sistema cubrirá el proceso completo de comercialización de vehículos usados dentro de una concesionaria.

Incluye:

- Recepción de vehículos
- Tasación
- Control documental
- Preparación del vehículo
- Gestión de inversiones
- Publicación de unidades
- Gestión comercial
- Simulación de financiación
- Venta
- Transferencia
- Dashboard Gerencial
